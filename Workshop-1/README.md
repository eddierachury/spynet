## Resumen del proyecto

Spynet es una aplicación web que analiza la composición tecnológica de cualquier sitio web dado una URL. Detecta frameworks frontend/backend, CDN, consulta registros DNS, WHOIS, geolocalización de IP e historial en Wayback Machine.
